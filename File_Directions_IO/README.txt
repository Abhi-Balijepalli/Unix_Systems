To compile: gcc --std=gnu99 -o movie_by_year main.c
To run: ./movie_by_year

Note:
The directories will be created after choosing a file to process, not when you run the program.

Summary of program:
Program will find largest, smallest or take a custom user input file in the home directory, process the data and store it in a struct linked list. After the file has been processed, it will create a create a new directory with a random_id attached with year.txt(s) which includes movies from that year.

Permissions:
Directories: 750 or rwxr-x---
Files: 640 or rw-r-----
